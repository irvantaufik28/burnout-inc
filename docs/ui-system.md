# UI and Visual Identity

## Philosophy: Structured Calm
Burnout Inc uses a high-end dashboard aesthetic to represent the digital life of a modern developer. The UI is designed to be readable, professional, and slightly stressful (via high-contrast status changes).

## Visual Rules
- **Background:** Zinc-950 (Total dark).
- **Borders:** Zinc-800 or Zinc-900 (Subtle depth).
- **Typography:** Inter/Sans-serif (Modern, clean).

## State Visualization
- **Safe:** Emerald/Green.
- **Warning:** Yellow.
- **Critical:** Red + Pulse animation.
- **Action Required:** Emerald pulse (e.g., active task).

## Component Rules
- **Modular Panels:** Every section (Stats, Board, Tasks) is a self-contained panel with a consistent header style.
- **Progressive Disclosure:** Advanced details only appear when relevant (e.g., Active Contract details only appear when a project is accepted).
