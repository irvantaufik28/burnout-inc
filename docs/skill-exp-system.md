# Skill Experience and Expertise

## System Identity
The Skill Experience system tracks technical expertise. This progress is persisted across sessions, allowing you to master technologies over days or weeks of play.

## Domains
- Core Methodology: Coding, Design, Marketing.
- Tech Stack: Frontend, Backend, AI, DevOps, Mobile.

## Weighted EXP Distribution
EXP is distributed based on project requirements. Frontend-heavy projects give Frontend EXP.

## Learning Outside the Comfort Zone
Learning outside your current highest skill provides an EXP bonus, encouraging experimentation.

## Mastery Curve
Mastery (LV 60+) is slower than initial learning, requiring dedication.
