# Product Lifecycle System

## System Identity
The strategic heart of the game. Players transition from "Ideation" (Blueprint) to "Execution" (Building) and finally "Market Validation" (Launch).

## Gameplay Effects
- Blueprint Strategy: Synergy between Type, Market, and Focus determines base potential.
- Development Vitals: Quality and Bugs accumulate during the Build phase.
- Passive Income: Launched products generate recurring revenue based on User count and Quality.

## Synergy Logic (Hidden)
- Certain combinations (e.g., AI Tool + Startups + Innovation) provide massive multipliers.
- Misalignment (e.g., Enterprise + Speed) results in poor launch reception.

## Revenue Formula
dailyRevenue = users * (0.5 + backendFamiliarity / 200)
High Quality increases User growth, while high Bugs increase User decay.

## Viral Chance
A "Viral Hit" (Score 90+) triggers a massive initial user surge, accelerating the transition from freelance dependency to product sustainability.
