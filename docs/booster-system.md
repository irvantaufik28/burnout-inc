# Booster and Coping Mechanic System

## System Identity
The desperate measures of a stressed freelancer. Boosters represent unhealthy coping mechanisms used to survive aggressive deadlines. They provide short-term survival benefits but create long-term instability.

## Soft-Limit and Pacing
Following the **Dual Time Scale** philosophy, stimulants help you keep up with fast Project Time while slowly destroying your Human Time stability.

## Core Mechanics
- **Diminishing Returns:** Repeated usage reduces effectiveness.
- **Escalating Aftermath:** Negative conditions trigger upon effect expiry.
- **Stimulant Fatigue:** Abuse leads to chronic conditions (Overstimulated, Anxiety).

## UI Visualization
The Operational Protocol panel displays:
- Remaining cooldowns.
- Current tolerance levels.
- Global System Fatigue risk indicator.
