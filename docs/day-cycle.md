# Day Cycle & Economy

## System Identity
The temporal pace-setter of the game. Manages the transition between work cycles and survival costs.

## Gameplay Effects
- Daily Expenses: Deducts capital every midnight (00:00) for "Startup Life" costs.
- Recovery: Restores a baseline amount of Energy every new day.
- Random Events: 10% chance at day-start to trigger impactful events (Positive or Negative).

## Expenses Logic
- Base Cost: $50/day.
- DevOps Mitigation: High DevOps familiarity can reduce daily server costs by up to 50%.

## Random Events
- Laptop Overheating: -15 Focus.
- Twitter Mention: +50 Users to a random portfolio product.
- AWS Outage: -2 Quality to a random portfolio product.
