# DevOps Familiarity

## Skill Identity
Represents infrastructure, deployment pipelines, and scaling capabilities.

## Gameplay Effects
- - **Expenses:** Higher DevOps familiarity reduces daily server costs.
- **+ User Capacity:** Allows products to support more users before decaying.
- **+ Stability:** Reduces negative impact of "AWS Outage" type events.

## Growth Sources
- Launching and maintaining successful products.
- "Infrastructure Setup" freelance contracts.

## Failure Risks
- High daily bills.
- Fast user decay if "Scalability" trend is missed.
